## 整体结构

点评必须严格遵循以下固定结构，不可增减或调换顺序：

```
#[标题，用一句话概括餐厅的核心特点]

位置: [地址信息，若店铺链接有则参考，否则写具体位置]

环境: [参考图片和其他用户的评价生成，2-3句话即可，不要太长]

口味:
[第一个菜品评价段落]
[第二个菜品评价段落]
[第三个菜品评价段落]
...
```

## 标题规范

### 长度
- 严格 15-20 字

### 内容要求
- 必须包含餐厅最突出的 1-2 个特点
- 可以提及：餐厅类型、经营年限、核心特色、口味特点

### 正确示例
- "开了很多年的台湾餐厅，品质和服务都很不错"
- "新开的日料店，新鲜度和摆盘都很用心"
- "地道的川菜馆，麻辣鲜香分量足"

### 错误示例
- ❌ "好吃"（太短）
- ❌ "这是一家位于市中心的开了很多年的非常正宗的台湾风味餐厅推荐大家来尝试"（太长）

## 位置描述规范

### 内容来源
- 优先使用搜索获取的地址信息
- 如果没有具体地址，描述位置特点（商圈、地铁可达性等）

### 写作要求
- 简洁明了，一句话
- 包含关键定位信息

### 正确示例
- "距离地铁站5分钟步行距离，商场3楼"
- "市中心商圈，步行可达主要景点"
- "写字楼附近，午餐时段人很多"

## 环境描述规范

### 长度
- 严格 2-3 句话，不超过 60 字

### 内容要素
基于环境照片分析，包含：
- 装修风格（简约、复古、现代、传统等）
- 座位布局（宽敞、紧凑、私密、开放等）
- 灯光氛围（温馨、明亮、昏暗、浪漫等）

### 正确示例
- "装修简约温馨，座位宽敞舒适，适合朋友小聚"
- "复古怀旧风格，灯光柔和，适合约会"
- "开放式厨房设计，干净整洁，可以看到厨师操作"

### 错误示例
- ❌ "装修非常简约温馨，整体色调以米白色为主，座位摆放非常宽敞舒适，每个桌子之间都有适当的距离，灯光也很柔和，非常适合朋友小聚聊天，卫生状况也非常好，地上没有一点垃圾"（超过60字）

## 口味评价规范

### 格式
每个菜品单独一段：
```
[菜品名称]：[具体评价]
```

### 每段必须包含的 5 个要素

| 要素 | 说明 | 示例词汇 |
|------|------|---------|
| 视觉呈现 | 从照片分析颜色、摆盘 | 金黄、翠绿、深红、精致、层次分明 |
| 口感描述 | 具体的口感词汇 | 嫩、脆、滑、糯、软、酥、劲道、Q弹 |
| 味道层次 | 基础味道和层次感 | 甜咸辣酸鲜、前中后味变化、味道协调 |
| 温度 | 热度或冷度是否合适 | 滚烫、温热、适中、冰爽、清凉 |
| 推荐指数 | 1-5星，⭐️ emoji | ⭐️⭐️⭐️⭐️⭐️ |

### 长度
- 每道菜 40-60 字（含星级）

### 正确示例（口语化、有真实感）
```
三杯鸡：鸡肉挺嫩的不柴，酱汁很香，锅底还在冒泡，趁热吃很过瘾 ⭐️⭐️⭐️⭐️⭐️

卤肉饭：肥瘦比例刚好，不会太腻，就是饭给的有点少 ⭐️⭐️⭐️⭐️

珍珠奶茶：珍珠挺Q弹的，甜度正好，中规中矩吧 ⭐️⭐️⭐️
```

### 错误示例（AI味重、过度夸赞）
- ❌ "三杯鸡：鸡肉嫩滑不柴，酱汁浓郁香醇，口感层次分明，温度恰到好处 ⭐️⭐️⭐️⭐️⭐️"（四字成语堆砌，全是夸）
- ❌ "三杯鸡很好吃，鸡肉很嫩，味道很不错"（太短，缺少视觉、温度等要素）
- ❌ 超过 60 字的冗长描述
- ❌ 每道菜都给 4-5 星（不真实）

### 星级参考标准
- ⭐️⭐️⭐️⭐️⭐️：极力推荐，各方面都出色
- ⭐️⭐️⭐️⭐️：很好，值得一试
- ⭐️⭐️⭐️：中规中矩，不功不过
- ⭐️⭐️：有明显不足
- ⭐️：不推荐

## 禁止事项

### 内容
- ❌ 编造未在照片中出现的菜品
- ❌ 复制粘贴其他用户评价原文
- ❌ 无意义的形容词堆砌
- ❌ 过度使用网络流行语
- ❌ 字数过少或过于简略

### 格式
- ❌ 破坏预设的结构格式
- ❌ 跳过任何必需部分
- ❌ 超过各部分的字数限制

### 表达
- ❌ 过于主观夸张的语言
- ❌ 模糊不清的表达（如"味道不错"、"挺好的"）
- ❌ 与照片不符的描述

## 完整示例

```
#开了很多年的台湾餐厅，品质和服务都很不错

位置: 距离地铁站5分钟步行距离，商场3楼

环境: 装修挺温馨的，座位之间距离够大，适合朋友聚餐

口味:
三杯鸡：鸡肉挺嫩的不柴，酱汁很香，锅底还在冒泡，趁热吃很过瘾 ⭐️⭐️⭐️⭐️⭐️

卤肉饭：肥瘦比例刚好，不会太腻，就是饭给的有点少 ⭐️⭐️⭐️⭐️

珍珠奶茶：珍珠挺Q弹的，甜度正好，中规中矩吧 ⭐️⭐️⭐️
```

## 写作风格核心原则

点评要像一个普通人随手写的，不是文案策划精心打磨的：

1. **说人话**：用日常口语，"挺嫩的" 比 "嫩滑不柴" 好
2. **别全夸**：真实点评一定有好有坏，至少有"还行"的部分
3. **少用四字成语**："浓郁醇厚"、"鲜美可口" 是 AI 味重灾区
4. **可以有个人偏好**："我个人不太能吃辣"、"我比较喜欢咸甜口的"
5. **星级要有区分**：不要每道菜都高分

## 词汇对照表（书面体 → 口语化）

| 书面体（少用） | 口语化（多用） |
|---------------|---------------|
| 嫩滑不柴 | 挺嫩的，不柴 |
| 酱汁浓郁香醇 | 酱汁很香 / 酱给的足 |
| 口感层次分明 | 吃起来有层次 |
| 色泽诱人 | 看着就有食欲 / 卖相不错 |
| 摆盘精致 | 摆盘挺用心的 |
| 鲜美可口 | 挺鲜的 |
| 回味悠长 | 吃完嘴里还有味 |
| 分量充足 | 量挺大的 |
| 温度适宜 | 上菜挺快，还是热的 |
| 入口即化 | 一咬就化了 |
| 外酥里嫩 | 外面脆脆的，里面很嫩 |

### 可以用的"不完美"表达
- "量有点少"、"稍微有点咸了"、"等了挺久的"
- "中规中矩吧"、"没有特别惊艳"、"性价比一般"
- "我个人觉得偏甜了"、"不太能吃辣的慎点"
